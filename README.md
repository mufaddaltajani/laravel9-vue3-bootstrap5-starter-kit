## Laravel 9 starter kit with VueJS 3 and Bootstrap 5

---

`git clone https://github.com/mufaddaltajani/laravel9-vue3-bootstrap5-starter-kit.git`

`cd laravel9-vue3-bootstrap5-starter-kit`

`composer install`

`cp .env.example .env`

`php artisan key:generate`

`npm install`

`npm run dev`
